# Description
A blackout top-down shooter. The name (Project Umbra) is temporary and subject to change in the future.
* Additional notes to be added soon

# Controls (Subject to change)
* WASD - Move Character
* R - Reload Weapon
* F - Toggle Flashlight
* Left Click - Shoot Gun
